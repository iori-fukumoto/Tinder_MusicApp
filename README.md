## 機能
・匿名ログイン機能
・検索機能
・スワイプ機能
・音声再生機能
・お気に入りリスト表示機能
・他のユーザーのお気に入りリスト確認機能

## 使用したAPI・ライブラリ
・Firebase
・Apple Music API
・VerticalCardSwiper
・SDWebImage
・PKHUD
・Alamofire
・SwiftyJSON
